Hexo + Github搭建博客

发表于 2018-02-08 | 分类于 Hexo

写在前面

emmmmm,本来这篇文章应该在去年6月份就写的，由于各种原因（好吧其实是懒~）荒废了，半年之后在一次CTF中，看到了许多优秀大学生关于比赛的wp后发现他们平时没事都在写博客，不断总结进步。于是，在一个伟大的日子里，我开始重新配置博客，但是出现了一大堆报错，浪费了半天的时间也没有解决，所以我把原来那个博客的配置文件删了，重新开始耗时两天建立了这个新的博客，用于总结和学习。

环境准备

安装Node.js

这个没什么难的，一路next就能装好。
在命令行中输入验证是否配好,如果能显示版本号就代表环境配好了。

安装Git

关于Git的下载和安装，参考这篇博客
安装完成后，打开Git Bash设置名称和邮箱

安装Hexo

上述两步完成后，就可以开始安装Hexo，打开Git Bash

漫长的等待之后，Hexo就安装成功了。

Github的配置

  • 首先注册,登陆Github
  • 然后右上角的加号，创建代码库（Create a new repository）

配置SSH

本地生成公钥私钥

三次enter过后，会得到两个本地文件id_rsa、id_rsa.pub在C:\Users\用户名\.ssh用文本编辑器（不要用自带的记事本，切记）打开id_rsa.pub，复制全部内容，进入Github，Title的内容没要求，Key就是id_rsa.pub中的内容，然后Add SSH key

初始化博客

  1. 选择一个目录，新建Blog文件夹（这个文件夹就是以后博客的配置文件夹）

  2. Git Bashcd到这个目录后，执行hexo init blog，漫长等待后提示成功

  3. 这个时候我们就可以看看初步的成果了，继续执行以下命令

在浏览器中输入localhost:4000,就可以看看博客初始的样子了。
