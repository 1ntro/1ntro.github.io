Kali-Linux学习笔记-1

发表于 2018-05-30 | 分类于 Kali-Linux

Linux系统命令

ls 显示文件或文档

-l 列出文件详细信息

-a 列出当前目录下所有文件及目录，包括隐藏的文件

mkdir 创建目录

-p 创建目录，若无父目录，则创建p

cd 切换目录

pwd 查看当前路径

touch 创建空文件（注意文件）

echo 创建带有内容的文件

echo 内容 > 文件 （覆盖）

echo 内容 >> 文件 （叠加）

cat 查看文件内容

cp 复制文件

mv 移动或重命名文件

more、less 查看文件(q退出）

rm 删除文件

-r 递归删除 可以删除子目录及文件

-f 强制删除

rmdir 删除空目录

find 在文件系统中搜索某文件

wc 统计文本中行数、字数、字符数

grep 在文本文件中查找某个字符串

head、tail 显示文件头、尾内容

Linux服务命令

service (service) start 启动指定的系统服务service

service (service) restart 重启指定的系统服务service

service (service) stop 停止指定的系统服务service

service （service) status 查看某项服务是否开启

update-rc.d (服务) enable 将某项服务加入开机启动

update-rc.d (服务) disable (level) 将某项服务移出开机启动，设置启动的优先级level(可选)

Linux文本编辑命令

vim使用

vim三种模式：命令模式、插入模式、编辑模式

:q 退出

:q! 强制退出（不保存）

:wq 保存并退出

:set number 显示行号

:set nonumber 隐藏行号

/root 在文档中查找root，按n跳到下一个

nano命令

类终端图形化界面支持组合键操作

leafpad命令

以命令的形式打开文件，若不指定，则创建一个文件

Kali工具命令

通用格式：

工具名 参数 路径 链接 地址 端口

比如：sqlmap -u URL 参数 库 表 参数 字段

hydra 参数 文件 参数 文件 参数 地址 端口

ssh 用户名@地址

mysql -u 用户 -p

Linux用户和权限

用户及用户组管理

/etc/passwd 存储用户账号

/etc/group 存储组账号

/etc/shadow 存储用户账号的密码

/etc/gshadow 存储用户组账号的密码

useradd 用户名 只增加一个用户

userdel 用户名 删除一个用户

adduser 用户名 增加一个用户（自动创建home目录）

passwd root 给root设置密码

Linux文件权限

R 读 数值表示4

W 写 数值表示2

X 可执行 数值表示1

-rw-rw-r–分为四段，“-”表示普通文件，“l”表示链接，“d”表示目录。

chmod更改权限。

文件压缩和解压

tar命令

解压：tar zxvf FileName.tar

压缩：tar czvf FileName.tar DirName

添加标准用户

useradd -m username1 附加创建用户主目录

passwd username1

usermod -a -G sudo username1 附加到指定用户组

chsh -s /bin/bash username1 附加bash shell操作权限

Kali-Linux虚拟网络

ip地址划分

利用vm自带的虚拟网络编辑器

保留地址、私有地址

桥接网络

和物理机同段ip地址

NAT模式

NAT模式让虚拟机借助NAT（网络地址转换）功能，虚拟一个网段，容纳更多主机地址。

DHCP协议和主机

DHCP动态主机协议，在当前网络获得一个合法上网的ID。

DHCP攻击，针对DHCP服务器进行DHCP拒绝服务攻击，造成DHCP服务器无地址可发放。防御策略：绑定静态地址，或者使用私有地址。

NAT端口映射

netstat -an 查看当前开放的端口

虚拟网络编辑器=》NAT设置=》添加=》端口号、IP地址

虚拟机的端口映射到物理机，渗透测试可能会用到。

SSH相关

SSH相当于Linux的控制通道

SSH协议可以有效防止远程管理过程中的信息泄露问题，而且链接更加隐蔽，基于ssh的远程控制可以与公私秘钥相互结合，增强服务器的稳定和控制。

修改ssh的配置文件

修改/etc/ssh/sshd_config文件，配置端口和可访问的ip，修改PasswordAuthentication yes以及PermitRootLogin yes

设置开机自启动

添加开机自启动：update-rc.d ssh enable

关闭：update-rc.d ssh disabled
